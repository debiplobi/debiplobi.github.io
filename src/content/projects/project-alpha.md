Project Alpha represents a synthesis of modern web practices aimed at achieving a perfect Lighthouse score without sacrificing aesthetic quality.

## Tech Stack
- **Framework:** Astro
- **UI Library:** React
- **Styling:** Tailwind CSS

The main challenge was ensuring that the minimalist design didn't feel "empty" but rather "curated." This was achieved through precise control over typography and micro-interactions.
