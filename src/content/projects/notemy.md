Notemy is a full-stack note-storing application with end-to-end encryption, secure authentication, and privacy-focused backend APIs.

## Key Features
- **End-to-End Encryption:** Ensures user notes are securely stored and cannot be read by anyone else.
- **Backend Caching:** Implemented a backend caching system to reduce redundant database queries and improve response times.
- **Privacy-focused APIs:** Designed with security and privacy top of mind.
